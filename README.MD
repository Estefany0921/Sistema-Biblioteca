## Descripción del proyecto

El proyecto consiste en el desarrollo de un sistema de biblioteca que permita organizar, consultar y administrar información relacionada con libros, usuarios y préstamos. La idea principal es crear una aplicación funcional que facilite el control básico de una biblioteca, permitiendo registrar libros disponibles, gestionar usuarios y llevar seguimiento de los préstamos realizados.

Este proyecto se desarrollará desde cero hasta su publicación en un servidor, aplicando buenas prácticas de programación, organización de archivos, uso de GitHub y trabajo colaborativo.

## Objetivo general

Desarrollar un sistema de biblioteca funcional que permita la gestión básica de libros, usuarios y préstamos, con una interfaz clara, ordenada y fácil de utilizar.

## Objetivos específicos

- Crear una interfaz inicial para la presentación del sistema.
- Registrar libros con información básica como título, autor, categoría, año y estado.
- Permitir la consulta de libros disponibles en la biblioteca.
- Registrar usuarios que puedan solicitar préstamos de libros.
- Gestionar préstamos y devoluciones de libros.
- Organizar el código del proyecto en un repositorio de GitHub.
- Publicar el proyecto en un servidor para su visualización pública.

## Funcionalidades principales

### Gestión de libros

El sistema permitirá registrar, visualizar, editar y eliminar libros. Cada libro tendrá información como título, autor, categoría, año de publicación y estado, indicando si se encuentra disponible o prestado.

### Gestión de usuarios

El sistema contará con un apartado para registrar usuarios de la biblioteca. Cada usuario podrá tener datos básicos como nombre, documento, correo electrónico y teléfono.

### Gestión de préstamos

El sistema permitirá registrar préstamos de libros a los usuarios. También se podrá llevar control de la fecha de préstamo, fecha estimada de devolución y estado del préstamo.

### Consulta de libros

Los usuarios podrán consultar los libros disponibles en la biblioteca mediante una lista organizada. Se podrá buscar información por título, autor o categoría.

### Página de inicio

El sistema tendrá una página principal donde se presentará el nombre del proyecto, una breve descripción y un menú de navegación hacia las secciones principales.
