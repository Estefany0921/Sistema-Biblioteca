¿Qué permite hacer este sistema?

El proyecto busca reemplazar el control manual de una biblioteca por una solución web básica, funcional y fácil de utilizar.

01

Gestión de libros

Permite registrar libros con datos como título, autor, categoría, año de publicación y estado del ejemplar.

02

Registro de usuarios

Permite guardar la información básica de los estudiantes o personas que pueden solicitar libros en préstamo.

03

Control de préstamos

Facilita el registro de préstamos, fechas de devolución y estado de cada libro prestado.

Módulos principales

El sistema estará dividido en secciones para facilitar la navegación y el manejo de la información.

Libros Registrar, consultar y actualizar libros. Usuarios Registrar estudiantes o lectores. Préstamos Controlar entregas y devoluciones.

Objetivo del proyecto

Desarrollar un sistema web funcional para la gestión básica de una biblioteca escolar, aplicando conocimientos de HTML, CSS y JavaScript.

Tecnologías utilizadas

  • HTML5 para la estructura de las páginas.
  • CSS3 para el diseño visual y responsive.
  • JavaScript para la interacción del sistema.
  • GitHub para el control del proyecto.
